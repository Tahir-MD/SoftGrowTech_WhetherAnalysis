# SoftGrowTech_Weather_Data_Analysis

## 📊 Project Overview
This project analyzes historical weather data to understand weather patterns, focusing on temperature and humidity trends. The analysis includes data cleaning, statistical analysis, and various visualizations to identify patterns and relationships in weather data.

## 📁 Dataset
- **Source**: Kaggle - Weather History Dataset
- **File**: `weatherHistory.csv`
- **Time Period**: 2006-2016
- **Features**: Temperature, Humidity, Wind Speed, Pressure, Precipitation Type, etc.

## 🎯 Objectives
- Load and clean weather dataset using Python
- Analyze temperature and humidity trends
- Create meaningful data visualizations
- Identify patterns in weather data
- Present insights using various charts

## 🛠️ Technologies Used
- Python 3.x
- Pandas (Data manipulation)
- NumPy (Numerical operations)
- Matplotlib (Data visualization)
- Seaborn (Statistical visualization)

## 📊 Key Analyses Performed
1. **Data Cleaning**: Handling missing values, date formatting
2. **Statistical Analysis**: Mean, max, min of temperature and humidity
3. **Temperature Analysis**: 
   - Distribution analysis
   - Monthly trends
   - Yearly patterns
4. **Humidity Analysis**:
   - Distribution patterns
   - Monthly variations
5. **Relationship Analysis**:
   - Temperature vs Humidity correlation
   - Multi-variable correlation matrix

## 📈 Visualizations Created
- Temperature Distribution Histogram
- Humidity Distribution Histogram
- Monthly Temperature Trend Line Chart
- Monthly Humidity Trend Line Chart
- Temperature vs Humidity Scatter Plot
- Daily Temperature Patterns
- Monthly Boxplots for Temperature & Humidity
- Correlation Heatmap
- Yearly Temperature Bar Chart
